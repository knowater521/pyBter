# pyBter

bter.com API Python 2.5+ wrapper.

# Official Document

 - [API Version 1.0](https://bter.com/api)

# How to use pyBter

1. You must have Python 2.5+ first.

2. [Install PIP](https://pip.pypa.io/en/stable/installing/)

3. Install Python Lib: requests

```python
pip install requests
```
